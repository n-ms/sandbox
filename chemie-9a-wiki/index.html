🔑 Kernbegriffe auf einen Blick

  • Oktettregel → Atome streben 8 Außenelektronen an
  • Edelgaskonfiguration → stabile Elektronenanordnung wie ein Edelgas
  • Kugelwolkenmodell → vereinfachtes Modell für die Elektronenverteilung
  • Elektronenpaarbindung → zwei Atome teilen sich Elektronenpaare
  • Elektronegativität (EN) → wie stark ein Atom Bindungselektronen anzieht
  • Lewis-Formel → Schreibweise mit Valenzstrichen
  • Dipol → Molekül mit ungleich verteilten Ladungen
  • EPA-Modell → Elektronenpaar-Abstoßungs-Modell → Molekülform
